# 上野匠 炭火燒肉職人 — 幸運轉輪版 🔥🥩🎡

> 把便當做好，就是最重要的事。

一款直式手機網頁小遊戲（HTML5），主題為日式炭火燒肉便當店。玩家選肉、控制火力、看準熟度翻面，再把烤好的肉夾進便當盒出餐，完成訂單後進入**幸運轉輪抽獎台**抽獎。內建日式炭火背景音樂。專為 **LINE LIFF** 設計。

> 本專案為 **幸運轉輪版**：抽獎機制由轉盤（fortune wheel）取代原本的三軸拉霸。

## 🕹️ 玩法

1. **選肉**：五花肉 / 里肌肉 / 雞腿排 / 排骨。
2. **烤肉**：肉放上烤網後會依火力慢慢變色 —— 生肉 → 半熟 → **完美** → 烤焦。**每塊肉烤熟速度隨機不同**，需分別在不同時機翻面。
3. **翻面 / 出餐**：在「完美」熟度時按下燙金 **「夾進便當盒」**，肉會夾進便當（白飯＋配菜）。
4. **再來一次**：左下角紅金按鈕可隨時重新開始遊戲。
5. **抽獎**：完成訂單後進入「**上野燒肉抽獎台**」幸運轉輪，**一天限抽一次**，加權隨機轉到結果後顯示獎勵內容並提示截圖給官方客服核銷。

## ✨ 特色

- 直式 `op_bg` 背景疊加佈局，等比縮放置中、自動適配手機與桌機
- 中央品牌徽章、今日選單訂單框、吊牌式選肉、火力／熟度金球軌道（金球與凹槽精準對位）
- 烤網肉品四階段熟度即時變化，**每塊肉隨機速率**，製造隨機翻面手感
- 便當盒擬真配菜：白飯、黃蘿蔔泡菜、青菜柴魚、蝦仁炒蛋
- 燙金書法「夾進便當盒」、「再來一次」紅金圖片按鈕
- **幸運轉輪抽獎台**（六格轉盤、加權隨機、一天一抽、獎勵說明＋客服核銷提示）
- **背景音樂**無限循環（遊戲頁＋抽獎頁，右上角 🔊 可開關，靜音設定記憶）

## 🛠️ 技術

純前端：原生 **HTML / CSS / JavaScript**，無框架、無建置流程。圖片素材於 `assets/`。

```
index.html        # 遊戲入口
style.css         # 樣式（含 op_bg 疊加佈局）
game.js           # 遊戲邏輯（烤肉、隨機速率、背景音樂）
assets/           # 背景、肉品、配菜、UI 素材、bgm.mp3 背景音樂
slot/             # 幸運轉輪抽獎台（完成訂單後進入）
  ├ index.html
  ├ style.css
  ├ app.js        # 幸運轉輪、加權抽獎、一天一抽邏輯
  └ assets/       # 轉盤 SVG、六種獎項圖示、繼續接單按鈕
```

## 💻 本機執行
